# Quick Start Guide: Managing Prepainted Miniatures

## For Artists/Admins

### Step 1: Log In
Go to `/admin/login` and sign in with your credentials.

### Step 2: Access Product Management
Once logged in to the dashboard, click the **"Manage Products"** button in the header.

### Adding a Product
1. Click "Add New Product"
2. Fill in:
   - Product Name (e.g., "Elven Wizard")
   - SKU (unique code, e.g., "PREMADE-008")
   - Price (e.g., 25.00)
   - Original Price (e.g., 40.00)
   - Image URL (e.g., "/wizard.jpg" or a full URL)
   - Description (details about the miniature)
3. Click "Create Product"

### Editing a Product
1. Find the product in the grid
2. Click "Edit"
3. Change any fields
4. Click "Update Product"

### Deleting a Product
1. Find the product
2. Click "Delete"
3. Confirm deletion

## First Time Setup

If this is the first time using the system, you need to initialize the database:

1. Visit: `https://your-domain.com/api/admin/setup`
2. This creates the necessary database tables
3. You only need to do this once

## That's It!

All changes you make will be immediately visible on the `/premade` page where customers shop for prepainted miniatures.
